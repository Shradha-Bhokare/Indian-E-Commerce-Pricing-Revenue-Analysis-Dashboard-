# Indian E-Commerce Pricing & Revenue Analysis Dashboard

Interactive Tableau dashboard analyzing revenue trends, discount impact, category performance, and state-wise sales distribution in Indian e-commerce data.

## Tools Used
- Tableau

## Key Insights
- Identified top-performing categories and revenue trends
- Analyzed discount impact across product categories
- Visualized state-wise revenue distribution and sales event performance
